Politica de Privacidad

Esta politica explica los terminos bajo los cuales RODUGA INVERSIONES SA usa y protege la informacion de los usuarios al usar el sitio web. La empresa se compromete a la seguridad de los datos de sus usuarios. Cuando se solicita completar campos con informacion personal se asegura que solo se usara de acuerdo con este documento. Esta politica puede cambiar con el tiempo por lo que se recomienda revisar esta pagina de forma continua.

Informacion que se recopila

Se puede recopilar nombre correo electronico y datos demograficos. Tambien informacion necesaria para pedidos entregas o facturacion.

Uso de la informacion

La informacion se usa para dar mejor servicio mantener registro de usuarios y mejorar productos. Se pueden enviar correos con ofertas o novedades que el usuario puede cancelar en cualquier momento.

Seguridad

RODUGA INVERSIONES SA esta comprometida en mantener la informacion segura. Se usan sistemas avanzados y se actualizan de forma constante para evitar accesos no autorizados.

Control de informacion personal

RODUGA INVERSIONES SA no vendera ni distribuira la informacion sin consentimiento salvo orden judicial. La empresa puede modificar esta politica en cualquier momento.
